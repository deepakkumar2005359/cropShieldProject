# 🌾 CropShield

AI-powered crop disease detection web application with bilingual support (English & Tamil), drag-and-drop image upload, analysis history, and modern responsive UI.

---

## 🚀 Features

- 📸 Upload crop images using drag & drop
- 🤖 Simulated AI crop disease analysis
- 🌿 Supports multiple crops:
  - Rice
  - Tomato
  - Wheat
  - Cotton
- 🌐 Bilingual Interface
  - English
  - தமிழ் (Tamil)
- 📊 Disease severity detection
- 📉 Crop damage estimation
- 💡 Treatment & prevention recommendations
- 🕘 Analysis history tracking
- 🎨 Fully responsive modern UI using Tailwind CSS
- ⚡ Smooth animations and transitions
- 🗑️ Delete history records

---

# 🖼️ Preview

## Home Page

```bash
index.html
```
